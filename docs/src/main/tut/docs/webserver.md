## Sample Webserver


The `corbeans-corda-webserver` example project implements an alternative to Corda's default
Node webserver using modules of corbeans. Run the server passing command line arguments to configure the nodes
and browse [/swagger-ui.html](http://localhost:8080/swagger-ui.html) to view the available endpoints documentation
generated by swagger.
